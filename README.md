# 🌍 Tata Steel Carbon Emission & ESG Stress Testing Tool (Excel)

> An interactive Microsoft Excel-based Carbon Emission & ESG Stress Testing Tool designed to demonstrate how data analytics can support sustainability-focused business decision-making.

---

# 📖 Project Overview

This project is an interactive Excel model inspired by a manufacturing business scenario.

After completing Excel as part of my Data Analytics learning journey, I wanted to apply my skills to a real-world business problem instead of building another sales or HR dashboard.

Since I aim to build my career in **ESG, Carbon Accounting, Sustainability Analytics, Financial Modelling, and Data Analytics**, I chose to build an Excel-based Carbon Emission & ESG Stress Testing Tool.

**This project represents the beginning of my ESG learning journey.**

---

# 🎯 Project Objectives

The objective of this project is to demonstrate how Microsoft Excel can be used to:

- Analyse carbon emissions
- Perform ESG stress testing
- Compare multiple business scenarios
- Build interactive dashboards
- Support sustainability-focused decision making
- Present business insights visually
